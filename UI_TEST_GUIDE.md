## 🐛 Potential Issues to Check

### Issue 1: Images Not Loading
**Symptom:** Broken image icons in scene cards
**Check:**
1. Open browser dev tools (F12)
2. Go to Network tab
3. Look for failed image requests
4. URL should be: `/storage/media/project_27/scene_XX_placeholder.jpg`

**Fix if needed:** Backend needs to serve static files from storage directory

### Issue 2: Video Not Playing
**Symptom:** Video player shows but doesn't play
**Reason:** Using external sample video (Big Buck Bunny)
**Expected:** Video should play since it's a public URL

### Issue 3: "Generate Images" Button Still Visible
**Symptom:** Button shows even though images exist
**Expected:** Button should say "Regenerate Images" since images already exist

## 🔧 If Something Doesn't Work

### Frontend Not Loading
```bash
# Check if frontend is running
ps aux | grep "next dev"

# Restart if needed
cd frontend
npm run dev
```

### Backend Not Responding
```bash
# Check if backend is running
ps aux | grep uvicorn

# Restart if needed
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### Images Not Showing
```bash
# Check images exist
ls -la storage/media/project_27/

# Verify image count (should be 10)
ls storage/media/project_27/ | wc -l
```
